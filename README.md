## Acestone Jewellery Platform

### Overview
Acestone Jewellery is a platform for showcasing and selling jewellery products. It also includes an internal POS system to assist with sales and inventory management.

### Features
- Product catalog for jewellery items
- Clean and elegant design
- Internal POS system for managing sales
- Inventory tracking capabilities

### Key Systems Implemented
- Custom POS system
- Backend inventory management
- Secure data handling

### Tech Stack
- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript

### Note
Due to client confidentiality, the full source code is not publicly available.
